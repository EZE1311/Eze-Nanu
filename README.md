# Plan integral — Eze & Nadia

Plan de alimentación + ejercicio para los dos.

## Archivos

- **`index.html`** — Página web autocontenida (HTML + CSS embebido). Esto es lo que se ve cuando se publica con GitHub Pages.
- **`Plan_integral_Eze_Nadia.docx`** — Versión Word, para imprimir o editar.

## Cómo publicar en GitHub Pages

1. Subir este repo a GitHub (si no está ya).
2. En el repo: **Settings → Pages**.
3. En "Build and deployment":
   - Source: **Deploy from a branch**
   - Branch: `main` (o la que tengas) — folder: `/ (root)`
4. Guardar. En 1–2 minutos GitHub te da una URL del estilo `https://<usuario>.github.io/<repo>/`.
5. Esa URL es la página del plan. Bookmark y listo.
